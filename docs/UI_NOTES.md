# UI Notes

Este documento contiene todas las decisiones de interfaz del proyecto.

---

# Identidad

El juego debe sentirse:

- Premium
- Moderno
- Divertido
- Competitivo

Nunca infantil.

---

# Logo

Utilizar siempre:

assets/logo/logo-main.webp

No modificar.

---

# Estilo

Todo debe verse 3D.

Material brillante.

Sombras suaves.

Mucho volumen.

Nada plano.

---

# Botones

Grandes.

Bordes redondeados.

Animación al presionar.

Glow.

---

# Casilleros

Hexagonales.

3D.

Colores:

- Amarillo
- Azul
- Verde
- Naranja
- Rojo

Siempre con el mismo material.

---

# Avatares

Figuras coleccionables.

Cabeza grande.

Cuerpo pequeño.

Vista isométrica.

Mirando levemente hacia arriba.

No utilizar emojis.

No usar CSS.

Siempre assets PNG/WebP.

---

# Cartas

Flip 3D.

Sombras.

Mucho volumen.

---

# Dados

Animación física.

Rebote.

Rotación.

---

# Animaciones

Duración:

200ms - 350ms.

Utilizar Framer Motion.

Todo debe tener easing.

---

# Responsive

Prioridad:

1. Celulares.

2. Tablets.

3. Desktop.

---

# UX

Cada acción importante debe tener:

- Animación.
- Sonido.
- Feedback visual.

El usuario nunca debe preguntarse si una acción fue registrada.

---

# Regla principal

Si una pantalla parece una página web, debe rediseñarse.

Debe parecer un videojuego.
